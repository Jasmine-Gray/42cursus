# Minishell

## 概要

Minishellは、42 Schoolのプロジェクトとして開発された、bashライクなシェルの実装です。POSIXシェルの基本機能を参考にしながら、シンプルで理解しやすい設計を目指しています。

## プロジェクトの目的

1. **システムプログラミングの学習** - プロセス管理、シグナル処理、パイプ通信などのUNIXシステムの基礎を実践的に学ぶ
2. **パーサーの実装** - コマンドライン入力を解析し、適切に実行する仕組みを構築
3. **エラーハンドリング** - 様々なエッジケースに対応した堅牢なプログラムの作成

## 主な機能

### 実装済み機能 ✅

- **基本的なコマンド実行**
  - 外部コマンドの実行（PATHからの検索）
  - 絶対パス・相対パスでの実行
  
- **ビルトインコマンド**
  - `echo` （-nオプション対応）
  - `cd` （相対・絶対パス対応）
  - `pwd`
  - `export`
  - `unset`
  - `env`
  - `exit`

- **リダイレクション**
  - 入力リダイレクション `<`
  - 出力リダイレクション `>`
  - 追記リダイレクション `>>`
  - ヒアドキュメント `<<`

- **パイプ**
  - 複数コマンドのパイプライン `cmd1 | cmd2 | cmd3`

- **環境変数**
  - 変数展開 `$VAR`
  - 環境変数の設定・削除

- **クォート処理**
  - シングルクォート `'` （変数展開なし）
  - ダブルクォート `"` （変数展開あり）

### 未実装機能 ❌

- 論理演算子（`&&`, `||`）
- セミコロン区切り（`;`）
- ワイルドカード展開（`*`, `?`）
- コマンド置換（`$(...)`, `` `...` ``）
- バックグラウンド実行（`&`）

## プロジェクト構造

```
minishell/
├── src/
│   ├── lexer/      # 字句解析
│   ├── parser/     # 構文解析
│   ├── executor/   # コマンド実行
│   ├── builtin/    # ビルトインコマンド
│   └── utils/      # ユーティリティ関数
├── tests/          # テストスイート
├── docs/           # ドキュメント
└── includes/       # ヘッダファイル
```

## Quick Start

```bash
# コンパイル
make

# 実行
./minishell

# テスト実行
bash tests/test_working.sh  # 動作確認済みテスト
bash tests/test_todo.sh      # 修正予定の機能テスト
```

## 設計思想

1. **シンプルさ優先** - 複雑な機能よりも、基本機能の正確な実装を重視
2. **bashとの互換性** - 基本的な動作はbashと同じになるよう設計
3. **エラーハンドリング** - メモリリークやクラッシュを防ぐ堅牢な実装
4. **可読性** - 42のNorm規約に従いつつ、理解しやすいコード
