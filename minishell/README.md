# Minishell Documentation

このディレクトリには、Minishellプロジェクトの包括的なドキュメントが含まれています。

## 📚 ドキュメント一覧

### [Home.md](Home.md)
プロジェクトの概要、主要機能、クイックスタートガイド

### [Development-History.md](Development-History.md)
開発の歴史、各フェーズでの実装内容、学んだ教訓

### [Architecture.md](Architecture.md)
システムアーキテクチャ、設計思想、内部実装の詳細

### [Testing-Strategy.md](Testing-Strategy.md)
テスト戦略、テストスイートの構成、実行方法

### [Known-Issues.md](Known-Issues.md)
既知の問題、優先度、修正状況

## 🗺️ ナビゲーション

```
初めての方 → Home.md
    ↓

    ↓
技術詳細を知りたい → Architecture.md
    ↓

    ↓

```

## 🎯 ドキュメントの目的

1. **プロジェクトの透明性** - 開発過程と決定を記録
2. **知識の共有** - 学んだことを文書化
3. **保守性の向上** - 将来の開発者への引き継ぎ
4. **品質保証** - テストとバグの体系的管理
